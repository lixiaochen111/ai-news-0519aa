## 🤔 分配建议的理由

### **为什么这4个放Tier 0？**

1. **Figma Blog**：设计师的核心工具，任何更新都值得关注
2. **OpenAI/Anthropic/Google AI**：AI行业三大巨头，产品发布必看

**关键特征：**
- ✅ 官方一手源，内容可信度100%
- ✅ 更新频率低（月更或双周更），不会霸屏
- ✅ 每条都是重磅更新，几乎无水文
- ✅ 不需要AI判断质量（官方发布=高质量）

---

### **为什么这7个放Tier 1？**

**共同特征：**
- ✅ 内容质量高，但不是所有都与"AI+设计"直接相关
- ✅ 比如UX Collective会有"色彩理论"、"用户研究"等非AI话题
- ✅ 需要AI判断：这篇文章是否对"关注AI设计工具的设计师"有价值

**为什么跳过关键词？**
- 避免误杀：UX Collective的"这个设计工具改变了我的工作流"可能没提"AI"，但实际是讲Figma AI功能
- 信任基础：这些源质量高，即使无AI关键词，内容也值得AI分析

---

### **为什么这5个放Tier 2？**

**共同特征：**
- ⚠️ 内容更杂，噪音比例更高
- ⚠️ 掘金：技术文章多，很多纯代码无关设计
- ⚠️ Webdesigner News：聚合源，转载多，需关键词先过滤
- ⚠️ 秋芝2046：B站UP主，视频标题可能不准确
- ⚠️ Newsletter：周报汇总，可能包含各类内容

**为什么需要关键词？**
- 成本控制：如果不先关键词筛选，AI要分析的内容太多
- 掘金可能每天50篇文章，但只有5篇真正相关

## 🔧 您可以调整的参考维度

在重新分配时，可以考虑这些因素：

### **1. 更新频率 vs Tier选择**
```
高频源（每天5+篇）→ 建议Tier 1或Tier 2，避免霸屏
低频源（每周/月更）→ 可以放Tier 0，不会淹没其他内容
```

### **2. 内容可控性**
```
官方源（自己的产品更新）→ Tier 0
高质量媒体（有编辑把关）→ Tier 1
UGC/聚合源（用户生成内容）→ Tier 2
```

### **3. 语言 vs 用户群体**
```
如果您的用户主要是中国设计师：
- 少数派、优设网可以考虑提升到Tier 0？
- 因为中文内容对他们更实用
```

### **4. 个人偏好**
```
如果您特别信任某个源的编辑眼光：
- 比如觉得Sidebar选的5条永远都很精准
- 可以考虑提升到Tier 0
```
